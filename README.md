# prompt-forge

## Claude Skill para generar prompts production-ready para cualquier herramienta de IA.

**v0.1.0** — initial release

---

## ¿Qué es un Claude Skill?

Un Claude Skill es un conjunto de instrucciones persistentes y configurables que extienden las capacidades de Claude.ai. Se instala una sola vez y está disponible en todas las conversaciones del usuario, actuando como una capa de especialización sin modificar el modelo base.

## Origen

prompt-forge evolucionó desde **PromptCL v2.0**, un custom GPT exclusivo para Claude que automatizaba la generación de prompts con triaje de criticidad, scoring de especificidad y selección de técnica por árbol de decisión binario. La migración a Claude Skill amplió el soporte a más de 30 herramientas de IA, manteniendo el núcleo de reglas anti-fabricación y el árbol de decisión explícito.

## Diferenciadores clave

- **Multi-tool routing (30+ herramientas):** Claude, GPT, o3/o4-mini, Gemini, Qwen, DeepSeek-R1, Ollama, Claude Code, Cursor, Midjourney, Stable Diffusion, ComfyUI, Sora, Runway, ElevenLabs, Zapier y más.
- **Reglas anti-fabricación duras:** Excluye explícitamente Mixture of Experts, Tree of Thought, Graph of Thought, Universal Self-Consistency, Meta-Prompting con expertos simulados y Prompt Chaining embebido como instrucción.
- **Selección de técnica por árbol de decisión:** Árbol binario de 8 nodos explícito, no basado en inercia ni en heurísticas vagas.
- **Detección nativa de modelos con razonamiento:** Sin Chain-of-Thought en o3, o4-mini, DeepSeek-R1 ni Qwen3-thinking.

## Instalación

1. Descarga `dist/prompt-forge.skill` desde [Releases](https://github.com/csanmarting/prompt-forge/releases) o directamente del repositorio.
2. Ve a **Claude.ai → Settings → Skills**.
3. Haz clic en **Upload** y selecciona el archivo `.skill`.
4. Activa el skill en tu workspace.

## Ejemplo de uso

**Solicitud cruda del usuario:**
> Necesito un prompt para que una IA analice los logs de nuestro ERP y encuentre inconsistencias en las transacciones de ventas del último trimestre.

**Prompt generado por prompt-forge:**
> Analiza los registros de transacciones de ventas del ERP del último trimestre (Q1 2026). Identifica: (1) transacciones con monto > desviación estándar de 3σ respecto a la media histórica, (2) órdenes duplicadas por número de cliente y fecha, (3) cambios de estado anómalos (ej. facturado → cancelado sin reverso), y (4) discrepancias entre el subtotal y la suma de ítems. Para cada hallazgo, reporta: ID de transacción, tipo de anomalía, severidad (crítica/alta/media/baja), y recomendación de acción. Formato de salida: tabla Markdown con columnas [ID, Tipo, Severidad, Recomendación], seguida de un resumen ejecutivo de 5 líneas máximo.

## What is a Claude Skill?

A Claude Skill is a persistent, configurable set of instructions that extends Claude.ai capabilities. Installed once, it is available across all user conversations, acting as a specialization layer without modifying the base model.

## Origin Story

prompt-forge evolved from **PromptCL v2.0**, a Claude-only custom GPT that automated prompt generation with criticality triage, specificity scoring, and binary decision-tree technique selection. The migration to Claude Skill format expanded support to 30+ AI tools while preserving the core anti-fabrication rules and explicit decision tree.

## Key Differentiators

- **Multi-tool routing (30+ tools):** Claude, GPT, o3/o4-mini, Gemini, Qwen, DeepSeek-R1, Ollama, Claude Code, Cursor, Midjourney, Stable Diffusion, ComfyUI, Sora, Runway, ElevenLabs, Zapier, and more.
- **Anti-fabrication hard rules:** Explicitly excludes Mixture of Experts, Tree of Thought, Graph of Thought, Universal Self-Consistency, Meta-Prompting with simulated experts, and embedded Prompt Chaining.
- **Decision-tree technique selection:** Explicit 8-node binary tree, not inertia-based or vague heuristics.
- **Reasoning-native model detection:** No Chain-of-Thought on o3, o4-mini, DeepSeek-R1, or Qwen3-thinking.

## Installation

1. Download `dist/prompt-forge.skill` from [Releases](https://github.com/csanmarting/prompt-forge/releases) or directly from the repo.
2. Go to **Claude.ai → Settings → Skills**.
3. Click **Upload** and select the `.skill` file.
4. Activate the skill in your workspace.

## Usage Example

**Raw user request:**
> I need a prompt for an AI to analyze our ERP sales logs and find inconsistencies in last quarter transactions.

**Prompt generated by prompt-forge:**
> Analyze Q1 2026 ERP sales transaction logs. Identify: (1) transactions > 3σ from historical mean, (2) duplicate orders by customer ID and date, (3) anomalous state transitions (e.g. invoiced → cancelled without reversal), and (4) subtotal vs. line-item sum discrepancies. For each finding, report: transaction ID, anomaly type, severity (critical/high/medium/low), and recommended action. Output: Markdown table [ID, Type, Severity, Recommendation] followed by a 5-line executive summary max.
